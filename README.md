# expressjs-boilerplate

## Requirements

### System requirements

* node >= 12
* npm
* (optional) WSL2
* (optional) Docker


### VSCode recommended plugins

* EditorConfig for VS Code
* ESLint
* TSLint


## Development 

1. Create an `.env` file form the example: `cp .env.example .env` 
2. Install dependencies `npm install`
2. Run the application `npm start`

## Test

`npm run test`
